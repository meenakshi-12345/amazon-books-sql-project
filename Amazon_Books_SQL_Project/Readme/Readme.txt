Amazon Books Data Analysis using SQL

A beginner-to-intermediate SQL practice project based on an Amazon Books dataset downloaded from Kaggle.
This project focuses on solving real-world business questions using SQL queries and analytical thinking.

The project includes:

Data exploration
Filtering and sorting
Aggregations
GROUP BY & HAVING
Subqueries
Business-oriented analysis
📌 Project Objective

The goal of this project is to practice SQL concepts by analyzing book sales and bestseller-related data from Amazon Books.

Using SQL queries, this project answers questions such as:

Which books are most reviewed?
Which publishers dominate the market?
Are expensive books highly rated?
Which categories generate maximum engagement?
